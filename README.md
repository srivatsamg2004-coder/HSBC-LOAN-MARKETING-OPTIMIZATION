# HSBC-LOAN-MARKETING-OPTIMIZATION
HSBC Bank Personal Loan Acceptance Analysis
HSBC Bank Personal Loan Acceptance Analysis & Marketing Optimization
Project Overview

This project analyzes HSBC customer data to identify customers most likely to accept personal loan offers. An interactive Excel dashboard was developed to support marketing decisions and improve campaign effectiveness.

Objectives
Analyze customer behavior
Improve loan campaign conversion
Reduce marketing costs
Identify high-potential customers
